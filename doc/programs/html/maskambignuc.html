Function

Masks all ambiguity characters in nucleotide sequences with N

Description

maskambignuc replaces any ambiguity codes in the input sequence with 'N'.

Usage

Here is a sample session with maskambignuc 


% maskambignuc 
Masks all ambiguity characters in nucleotide sequences with N
Input (gapped) nucleotide sequence(s): ambignuc.fasta
output sequence(s) [ambignuc.fasta]:

Input file format

maskambignuc reads any normal sequence USAs.

Input files for usage example

File: ambignuc.fasta

>AMBIGNUC
ABCDGHKMNRSTUVWY
abcdghkmnrstuvwy

Output file format

maskambignuc rewrites the input sequence.

Output files for usage example

File: ambignuc.fasta

>AMBIGNUC
ANCNGNNNNNNTUNNNancngnnnnnntunnn
