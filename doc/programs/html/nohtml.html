Function

Remove mark-up (e.g. HTML tags) from an ASCII text file

Description

Removes all HTML tags from the input sequence. No whitespace is inserted and no attempt is made to provide special processing for any of the tags.

Usage

Here is a sample session with nohtml 


% nohtml 
Remove mark-up (e.g. HTML tags) from an ASCII text file
ASCII text file: webfile.html
ASCII text output file [webfile.nohtml]:
