Function

Get sequence variations

Description

variationget reads sequence variation data and writes the data in one of the EMBOSS variation output formats.

Usage

Here is a sample session with variationget 


% variationget valid-4.0.vcf 
Get sequence variations
Variation output file [0.vcf]: test.vcf
