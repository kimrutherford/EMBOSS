Function

   Read and write (return) many sets of sequences

Description

   seqretsetall reads and writes multiple sets of sequences.

Usage

   Here is a sample session with seqretsetall


% seqretsetall -osf phylip
Read and write (return) many sets of sequences
Input sequence set(s): globins-all.phy
output sequence(s) [globins-all.phylip]:
